# Themes

Complete user-selectable theme definitions and their PNG previews.

## Theme rules

- Each theme is a complete visual opinion: colors, typography, density, spacing, widths, radii, and common component values.
- Do not split typography or density into separate toggles.
- Theme definitions should be original work inspired by broad visual characteristics, not copied from third-party CSS.
- Include appropriate UI, content, and monospace font choices in every theme.
- Keep one `.ts` definition and one `.png` preview per theme.
- Use the shared contract from `../core`; do not add theme-specific application logic here.

The initial set will include Light Brew, Dark Brew, Notation Paper, Notation Ink, Cupertino Light, Cupertino Dark, Encyclopedia Light, Encyclopedia Dark, Atom-Light, Atom-Dark, Storm, Night, Flash, and Hacker.
